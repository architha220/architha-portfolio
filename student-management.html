Smart Student Management System

Smart Student Management System is a Python-based application developed to efficiently manage student records using CRUD operations, searching, sorting, and file handling techniques through a menu-driven command-line interface.

Technologies Used

  • Python
  • File Handling
  • CSV / Text Files
  • Command Line Interface
  • VS Code

Main Features

  • Add Student Records
  • Update Student Information
  • Delete Student Records
  • Search Student Details
  • Sort Student Data
  • Marks & Grade Calculation
  • File Storage Management
  • User-Friendly CLI Interface
