CIRC – Conversational Image Recognition Chatbot

CIRC is an AI-based web application designed to make image understanding interactive through conversational AI. Users can upload images and ask questions related to image content using a chatbot-style interface.

Technologies Used

  • Python
  • Flask
  • HTML
  • CSS
  • JavaScript
  • MongoDB
  • Gemini API

Main Features

  • Image Upload & Analysis
  • Conversational AI Interface
  • Real-Time Responses
  • AI Image Understanding
  • MongoDB Integration
  • Responsive Web Interface
